# ✈️ Trip Planner

> A full-stack web application for creating and managing travel plans with user authentication, city autocomplete, and REST API integration.

## 🎨 Frontend

- ⚛️ React
- 📘 TypeScript
- ⚡ Vite
- 🗂️ Redux Toolkit
- 🔐 Authentication
- 🌐 REST API
- 📍 OpenTripMap API
- 🧭 React Router
- 🎨 SCSS

## 🖥️ Backend

- 🟢 Node.js
- 🚂 Express.js
- 📘 TypeScript
- 🗄️ Supabase
